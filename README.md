# enWrite Desktop
### Status
Alpha, Does not Persists Data yet.

### Run
1. Clone the Project, go to Directory/build/libs/
2. Run the FXApp-1.0-SNAPSHOT.jar in JRE
3. Using   ``` java -jar FXApp-1.0-SNAPSHOT.jar```

#### Or
1. Clone and Run using Gradle
2. ``` ./gradlew build ```
3. ``` ./gradlew run ```
